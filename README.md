# SSGD

## Overview

Implementation of Stochastic subgradient descent method.

Analysis using python notebook, with numpy for numerical analysis

## Features
- si distribution over convergent rate
- In-Sample vs Out-Sample error


## Installation method

1. Install uv: https://docs.astral.sh/uv/getting-started/installation/
2. run `uv sync`
3. run main.ipynb
